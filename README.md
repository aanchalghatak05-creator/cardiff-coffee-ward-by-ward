# Cardiff Coffee · Ward by Ward ☕

**Live Project:** [https://aanchalghatak05-creator.github.io/cardiff-coffee-ward-by-ward](https://aanchalghatak05-creator.github.io/cardiff-coffee-ward-by-ward)

An interactive scrollytelling data journalism project mapping Cardiff's independent café scene, ward by ward. Built as part of the MSc Computational and Data Journalism programme at Cardiff University.

---

## 📖 About the Project

Cardiff has a thriving independent café culture — but how is it distributed across the city's wards? This project combines geographic data, interactive mapping, and scrollytelling narrative to explore where independent cafés cluster, which wards are underserved, and what that tells us about Cardiff's neighbourhoods.

The piece is designed for a general audience and presents data-driven findings through an immersive, scroll-driven experience.

---

## ✨ Features

- **Interactive Leaflet.js map** — explore Cardiff's independent cafés plotted by ward
- **Scrollytelling narrative** — story-driven format that guides the reader through key findings
- **Ward-by-ward breakdown** — geographic analysis of café density and distribution
- **Responsive design** — works across desktop and mobile
- **Custom typography and styling** — editorial design built for readability and visual impact

## 🗺 Data

Geographic and café data was collected, cleaned, and structured as part of the MSc group research process. Data covers Cardiff's ward boundaries and the locations of independent (non-chain) cafés across the city.
